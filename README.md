# Scientific Calculator Android App

## Prerequisites
- Android Studio (Latest version recommended)
- JDK 8 or higher
- Android SDK

## Setup and Run

1. Clone the project repository
2. Open Android Studio
3. Select "Open an Existing Project"
4. Navigate to the project directory and select the root folder
5. Wait for Gradle sync to complete
6. Connect an Android device or set up an Android Emulator
7. Click the "Run" button (green play icon) or press Shift+F10

### Troubleshooting
- Ensure you have the latest Android SDK tools installed
- Check that your device/emulator meets the minimum SDK version (23)
- Verify Gradle dependencies are correctly downloaded

## Features
- Scientific calculations
- Programming mode calculations
- Basic and advanced mathematical operations
